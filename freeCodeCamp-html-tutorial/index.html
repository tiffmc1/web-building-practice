HTML4 vs HTML5

HTML5 is the current version of the language recommended for use since 2014

It comes with added features such as:

  • Cleaner code structure and improved semantics
  • Simpler DOCTYPE declaration; <!DOCTYPE html>
  • Cross-browser compatibility
  • More elements! For example, audio and video tags. HTML4 could not support media this way, which led to using third-party browser plug-ins, such as Adobe Flash
How does HTML5 benefit the user?

HTML5 gave the developer more control over the performance of their websites. Many “hacks” commonly used in HTML4 are now inherent to the language, which allows for a faster and smoother user experience
